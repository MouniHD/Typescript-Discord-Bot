# Typescript-Discord-Bot

` A Simple Typescript Discord Bot`<br>

# Config

`The Config is on src/utils/config` <br>

`export const token: string = "token";` <br>

`npm i @types/node`<br>
`npm i -g typescript`<br>
`npm i discord.js`<br>
`npm i discord-akairo`<br>
`npm i path`<br>
